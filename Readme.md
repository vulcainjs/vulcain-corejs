## Generate Typedoc  
Generate Typescript documentation  
Launch :  
```bash 
$ gulp doc
```

After launch web-server for visualize documentation on **docs** folder

If you haven't **live-server** installed, launch 
```bash 
$ npm install -g live-server
```


Visualize :
```bash 
$ live-server ts-doc
```
